# Background Info

## What is this?
This codespace is part of an ongoing project called Code Quiz Builder(CQB) that started in January 2026. This project is about creating a website that can create quizzes for users to test them on different kinds of programming topics. Here are what the folders and files at the root of this codespace do:

images: Stores the images and favicon icons that the platform folders use.

platform-v1: The main folder of this project that holds the code and files to display version 1 of the CQB website. Open the main.html file to view the frontend.

platform-v2: A folder prepared to contain code and files of version 2 of this project. Currently it has basic files, but will be developed over time and eventually become the main folder.

scratch: This folder holds code and logic snippets used for testing ideas, and is not used directly for the CQB.

.gitignore: this file prevents certain files from being committed and push to github

resources.txt: contains links to videos and websites useful for this project

## Setup
Run this command in terminal to install the right dependencies for the code to work:

npm install

If you want to open the website in platform-v2, execute the command below. You should get the link:

node question-builder.js
